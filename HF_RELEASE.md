### Naming rationale

- **`35b` → `18b`**: After surgery, each model retains exactly 128 of 256 experts per layer.
  Total parameter count drops from 35B → ~17.5B ≈ **18B**. Active parameters per token
  are unchanged at 3B (`a3b`).
- **`_T` suffix dropped**: All 8 releases are text-only pruned variants. The `_T` suffix
  is an internal modality discriminator; it is omitted from public names to avoid confusion.
  Vision capability requires a separate pipeline not included in this release.
- **`coding` not `systems`**: Internally the algo/systems model uses the `SYSTEMS` corpus
  tag (C++/Go/Rust/PowerShell/SQL/OS). It is published as `coding` for discoverability.
  The COD_ALL blended mask was evaluated and rejected due to measurable task-level
  performance degradation from sub-domain dilution (see RESEARCH_LOG.md §9.3).

## 4. Upload Commands

Requires: `pip install huggingface_hub` and `huggingface-cli login`

```powershell
# One-time login
huggingface-cli login

# Per model — substitute {DOMAIN} for each
$ORG   = "JThomas-CoE"
$REPOS = @(
    @{ repo="coe-qwen3.5-coding-18b-a3b";      gguf="CoE-SYSTEMS-35b-A3b-K128-q4_K_M.gguf";  pub="CoE-qwen3.5-coding-18b-a3b-Q4_K_M.gguf";      mask="coverage_SYSTEMS_K128.pt" },
    @{ repo="coe-qwen3.5-web-18b-a3b";          gguf="CoE-WEB_T-35b-A3b-K128-q4_K_M.gguf";    pub="CoE-qwen3.5-web-18b-a3b-Q4_K_M.gguf";          mask="coverage_WEB_T_K128.pt" },
    @{ repo="coe-qwen3.5-math-18b-a3b";         gguf="CoE-MATH_T-35b-A3b-K128-q4_K_M.gguf";   pub="CoE-qwen3.5-math-18b-a3b-Q4_K_M.gguf";         mask="coverage_MATH_T_K128.pt" },
    @{ repo="coe-qwen3.5-physics-18b-a3b";      gguf="CoE-PHYS_T-35b-A3b-K128-q4_K_M.gguf";   pub="CoE-qwen3.5-physics-18b-a3b-Q4_K_M.gguf";      mask="coverage_PHYS_T_K128.pt" },
    @{ repo="coe-qwen3.5-biology-18b-a3b";      gguf="CoE-BIO_T-35b-A3b-K128-q4_K_M.gguf";    pub="CoE-qwen3.5-biology-18b-a3b-Q4_K_M.gguf";      mask="coverage_BIO_T_K128.pt" },
    @{ repo="coe-qwen3.5-engineering-18b-a3b";  gguf="CoE-ENG-35b-A3b-K128-q4_K_M.gguf";      pub="CoE-qwen3.5-engineering-18b-a3b-Q4_K_M.gguf";  mask="coverage_ENG_K128.pt" },
    @{ repo="coe-qwen3.5-vocational-18b-a3b";   gguf="CoE-VOC-35b-A3b-K128-q4_K_M.gguf";      pub="CoE-qwen3.5-vocational-18b-a3b-Q4_K_M.gguf";   mask="coverage_VOC_K128.pt" },
    @{ repo="coe-qwen3.5-humanities-18b-a3b";   gguf="CoE-HUM-35b-A3b-K128-q4_K_M.gguf";      pub="CoE-qwen3.5-humanities-18b-a3b-Q4_K_M.gguf";   mask="coverage_HUM_K128.pt" }
)

$STAGED   = "D:\ollama\staged"
$MASKDIR  = "c:\RyzenAI\college of experts\qwen3.5\masks\specialist"

foreach ($m in $REPOS) {
    $repo = "$ORG/$($m.repo)"
    Write-Host "`n=== $repo ===" -ForegroundColor Cyan

    # Create repo (idempotent)
    huggingface-cli repo create $m.repo --type model --yes

    # Upload GGUF (rename to public convention on upload)
    huggingface-cli upload $repo "$STAGED\$($m.gguf)" "$($m.pub)"

    # Upload mask
    huggingface-cli upload $repo "$MASKDIR\$($m.mask)" "masks/$($m.mask)"

    # Upload Modelfile
    huggingface-cli upload $repo "$STAGED\Modelfile-$($m.gguf.Replace('.gguf','.txt'))" "Modelfile"

    # Upload README (must be prepared per-model from template first)
    huggingface-cli upload $repo ".\hf_readmes\$($m.repo)\README.md" "README.md"
}
```

**Before running:** prepare per-model READMEs from `scripts/hf_readme_template.md` into
`hf_readmes/{repo-name}/README.md`.
